# Videos Directory

This directory should contain your demo and technical walkthrough videos.

## Required Videos

1. **Demo Video** - A non-technical demonstration of the system
   - Shows the web interface or CLI in action
   - Demonstrates generating a cover letter from resume + job description
   - Should be accessible to non-technical audiences

2. **Technical Walkthrough Video** - A detailed technical explanation
   - Walkthrough of the codebase and architecture
   - Explains the ML pipeline, training process, and evaluation
   - Should demonstrate understanding of the implementation

## Video Requirements

- Videos should be in a common format (MP4 recommended)
- Keep file sizes reasonable for repository hosting
- Consider uploading to YouTube/Vimeo and linking in README.md if files are too large
